## 文件结构
- `MEMORY.md` — AI教练长期记忆（数据分析结论、关键指标）
- `USER.md` — 个人信息与训练偏好
- `garmin/` — Garmin 训练数据
  - `activities_summary.csv` — 104次训练记录（含multisport）
  - `weekly_stats.md` — 周训练统计
  - `sleep_data.csv` — 睡眠数据
  - `heart_rate_daily.csv` — 日心率数据
  - `max_heart_rate.csv` — 日最大心率
  - `endurance_score.csv` — 耐力得分趋势
  - `parse_fit.py` — FIT文件解析脚本
- `training/` — 训练计划
  - `current_plan.md` — 当前有效计划（待生成）
- `memory/` — 日志记录
  - `2026-03-22.md` — 今日会话记录
