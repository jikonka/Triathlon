### 技术问题讨论

**Q1：骑完换跑，前 0.5km 跑得非常快，心率飙高**
→ 这是"骑跑速度残留效应"（Cadence/Speed Carry-over）
→ 大脑记住了骑行节奏，切换跑步后潜意识保持高步频高速度
→ 解决方案：换项后**第一步就刻意减速**，前 200-400m 专注慢跑 + 深呼吸，让步频降到 170-175spm

**Q2：跑步心率控制在 Z3 时配速需要 7:30+，很慢怎么办**
→ 正常！骑行后跑步心率会虚高 5-10bpm，7:30 是真实 Z3 配速
→ 可以通过 **大量 Z2 跑**（2-3 次/周，每次 45-60min）来降低同配速心率
→ 时间线：6-8 周可见效，5/10 时应能提速至 7:00/km @ Z3

**Q3：能否增加更多 Z2 跑？跑步最便捷**
→ 完全可以，而且是明智选择
→ 条件：频率 3次/周以内，单次不超过 60min，步频控 170-175spm，整个过程心率严格≤129bpm
→ 雨天工作日早上/午间均可安排

**Q4：比赛心率区间规划（5/10 第一场）**
→ 🏊 游泳：不看心率，专注节奏和划频（水下难监测）
→ 🚴 骑行：目标 Z3（不超 Z4），HR ≤147bpm
→ 🏃 跑步：前 1km 克制在 Z2（≤129bpm），后 4km 可升到 Z3，最后 1km 可试探 Z4

**Q5：压心率有什么办法（不只是走路）**
→ 缩短步幅（跑步心率 80% 是步幅过大导致）
→ 鼻吸口呼放慢呼吸节奏
→ 头微低看地面 3-4m 处（减少竖向震动）
→ 手臂放松下沉（减少上身紧张消耗）

**Q6：需要吃胶吗**
→ Sprint 铁三总时长约 2h，**临界需要**（理论 60min 后补糖）
→ 第一场（5/10）建议：骑行 30min 时吃 1 颗，跑步起跑前再吃 1 颗
→ 先在训练中练习（今后砖式训练时带 1-2 颗能量胶试验）
